
\# User Management System (Spring Boot)



\## Project Description

This is a User Management System developed using Spring Boot.  

The application provides secure REST APIs for user registration, login, and role-based access using JWT authentication.



\## Features

\- User Registration \& Login

\- JWT-based Authentication

\- Role-based Authorization (USER / ADMIN)

\- RESTful APIs

\- Global Exception Handling

\- Secure password encryption (BCrypt)



\## Tech Stack

\- Java 8

\- Spring Boot

\- Spring Security

\- JWT

\- Spring Data JPA

\- Hibernate

\- MySQL

\- Maven



## Prerequisites (Software & Tools)

- Java: JDK 1.8 (Java SE 8)
- Spring Boot: 2.7.14
- IDE: Spring Tool Suite (STS) 3.19.x
- Database: MySQL 5.1.49
- Database Tool: MySQL Workbench 8.x
- Build Tool: Maven
- API Testing Tool: Postman
